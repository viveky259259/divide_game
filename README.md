# Flutter Game Project: Divide

## Feature
 1. [x] 3 x 3 board
 2. [x] 4 x 4 board
 3. [x] Point calculations
 4. [x] next and after next calculated values
 5. [x] Winner calculation
 6. [x] Timer (Play, pause)
 7. [x] Reset Game
 8. [x] Bg sound, Tap sound, clear sound
